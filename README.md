# Sales Dashboard - Power BI

## Project Overview
Bu dashboard satış məlumatlarının analizi üçün hazırlanıb.

## İstifadə olunan texnologiyalar
- Power BI
- DAX
- Power Query
- Excel

## Dashboard Features
- Total Sales KPI
- Region üzrə satış analizi
- Məhsul kateqoriyaları
- Aylıq trend analizi
- Slicer və filterlər
