# Sales Dashboard - Power BI

## Project Overview
This dashboard is designed for sales data analysis.

## Technologies used
- Power BI
- DAX
- Power Query
- Excel

## Dashboard Features
- Total Sales KPI
- Regional Sales Analysis
- Product Categories
- Monthly Trend Analysis
- Slicers and filters

## Dataset
The dataset was created for testing purposes.
